Installation des applications

Application Forfait

  • Copier le sous repertoire "reper" du repertoire "instal" sur un lecteur de disque c: ou d: etc...
  • Entrer dans le repertoire "reper' sur le disque local et créer un raccourci du fichier "a3h-forf" sur le bureau

    • Application Reel

  • Copier le sous repertoire "calreel" du repertoire "instal" sur un lecteur de disque c: ou d: etc...
  • Entrer dans le repertoire "calreel' sur le disque local et créer un raccourci du fichier "a3h" sur le bureau

    • Application Verification

  • Copier le sous repertoire "vp" du repertoire "instal" sur un lecteur de disque c: ou d: etc...
  • Entrer dans le repertoire "vp' sur le disque local et créer un raccourci du fichier "a3h-vp" sur le bureau

    • Dernières étapes

  • Lancer le fichier "setup" du sous repertoire setup du repertoire "instal"
  • Lancer le fichier "setup" du sous repertoire setupforfait du repertoire "instal"
  • Lancer le fichier "setup" du sous repertoire setupreel du repertoire "instal"
